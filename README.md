# Digital Agency Website

## About This Project
This project is a professional multi-page website built for a digital agency.
It includes Home, About, Services, and Contact pages with a responsive layout
and basic interactivity.

## Technologies Used
- HTML5 (Semantic structure)
- CSS Grid & Flexbox (Layouts)
- SASS (Style organization)
- JavaScript (DOM manipulation & form validation)

## Features
- Responsive design for desktop, tablet, and mobile
- Clean navigation across pages
- Mobile menu toggle
- Contact form validation
- Organized folder structure

## What I Learned
- Planning website structure before coding
- Building layouts using Grid and Flexbox
- Writing maintainable CSS using SASS
- Handling user interactions with JavaScript
- Testing and optimizing a website

## How to Run
Download the project and open `index.html` in a web browser.
